# TradeDNA 产品 PRD V1.1

> 2026-06-20 修订。V1.0 内容过浅，本版重写，明确定位为「内容获客工具」而非独立付费产品。

---

## 一、产品定位

**中文名**：交易人格测评
**英文名**：TradeDNA
**内容 IP**：交易DNA实验室
**slogan**：发现你的交易基因，找到最适合你的交易体系

**一句话定位**：通过 25 题交易人格测评，给散户/初阶量化用户一个"重新认识自己"的镜子，并作为「量化新手村」公众号 + 抖音 + 小红书的**长期获客钩子**。

**核心理念**：没有最好的交易系统，只有最适合你的交易系统。绝大多数散户亏钱的根因不是策略选错，而是**策略与性格错配**。

---

## 二、产品定位的关键澄清

### 它不是什么

- ❌ **不是一个付费产品**。测评类工具的复访率极低，C 端付费天花板很低(参考 MBTI、九型人格、盖洛普 CliftonStrengths——只有盖洛普靠 B 端培训活下来)。
- ❌ **不是荐股工具**。绝不涉及个股评价，全程规避证券投资咨询合规红线。
- ❌ **不是策略发布平台**。不输出"该买什么"，只输出"什么适合你"。

### 它是什么

- ✅ **公众号「量化新手村」的获客钩子**：测评结束引导关注公众号领取深度报告
- ✅ **抖音/小红书的话题素材库**：16 人格 × 16 名人对照 × N 条衍生选题 = 上百条短视频素材
- ✅ **作者 IP 的人设载体**：通过"我帮你看清你是谁"建立专业信任，反哺主业(明日当空 / 未来私募)
- ✅ **用户分层工具**：测评结果即用户标签，方便后续精细化运营

## 三、目标用户画像

### 主用户：A 股散户中的"反思型用户"(占比 60%)

- 30-45 岁，男性为主，月收入 1-3 万
- 入市 2-5 年，亏多赚少
- 已意识到"问题可能在自己身上"，开始寻求体系化方法
- 对 MBTI、九型人格等测评工具有基础认知

### 次用户：初阶量化爱好者(占比 30%)

- 25-40 岁，技术背景或金融背景
- 看过《量化投资》《股票作手回忆录》《海龟交易法则》
- 在 A 股摸索量化但缺少导师指引
- 是「量化新手村」公众号的精准目标人群

### 偶发用户：好奇心驱动型(占比 10%)

- 通过朋友分享或抖音引流进入
- 测完即走，是传播链上的中转节点

## 四、核心模型

### 四维交易人格模型

| 维度 | 代码 | 含义 |
|---|---|---|
| 决策方式 | A / B | 分析驱动 / 直觉驱动 |
| 交易周期 | S / L | 短线 / 长线 |
| 风险偏好 | C / G | 保守 / 进攻 |
| 执行能力 | D / F | 纪律 / 灵活 |

组合后生成 16 种交易人格，每种人格关联：
- 1 位**对照交易导师/投资人**(从西蒙斯到游资代表，覆盖中外、学院派与实战派)
- 4-6 条**核心特性**
- 4-6 条**典型风险**
- 5 种**推荐交易体系**及匹配度(★1-★5)
- 3 条**行动建议**

### 题目结构

| 题号 | 维度 | 性质 |
|---|---|---|
| Q1-Q5 | 决策方式 | 主测 |
| Q6-Q10 | 交易周期 | 主测 |
| Q11-Q15 | 风险偏好 | 主测 |
| Q16-Q20 | 执行能力 | 主测 |
| Q21-Q25 | **行为题**(原"校验题") | 模拟真实场景，权重 0.5，检测前 20 题答案与实际行为的一致性 |

行为题示例:"账户连续盈利 3 天后，第 4 天你的真实操作是？"——前 20 题问的是"你认为自己怎么想"，行为题问的是"你实际怎么做"。两者背离时报告页会给出"一致性提示"。

## 五、页面结构与核心闭环

```
首页 → 测评说明 → 25 题答题 → 结果报告 → 分享海报 → 引导关注公众号
```

| 页面 | 核心目标 | 关键元素 |
|---|---|---|
| 首页 | 建立专业感、激发好奇 | DNA 双螺旋动画、stat 行(25 题/5 分钟/16 人格)、开始按钮 |
| 测评说明 | 降低答题压力、设期望 | 4 维度卡片、测评输出清单、非荐股声明 |
| 答题 | 流畅完成 25 题 | 维度 pill、进度条、A/B 选项、上/下题 |
| 结果 | **传播触发点** | DNA 大码、人格名、雷达图、**名人导师卡**、核心特性、推荐体系、行动建议、四维解析 |
| 海报 | **关键传播载体** | 品牌卡 + DNA 码 + 人格 + QR 码 → 引流公众号 |

## 六、传播与增长机制

> 这一节是 V1.0 PRD 完全缺失的，也是产品成败的关键。

### 病毒系数(K 值)三要素

**1. 强标签性**
- 4 位 DNA 码(如 ASGD)易记、可对照、有圈层认同感(类比 INTJ/ENFP)
- 16 人格命名带画面感(量化猎手、龙头战将、梦想家…)

**2. 强分享冲动**
- 名人导师卡：让用户产生"原来我和西蒙斯/巴菲特/利弗莫尔是同类"的代入感
- 分享海报包含 DNA 码 + 人格 + 名人，截图即可传播
- (V1.5 规划)互测功能："你的搭档/师傅和你匹配度多少？"

**3. 强后续钩子**
- 结果页底部:"想看你这种人格的完整复盘报告？关注公众号回复 DNA 码领取"
- (V2 规划)16 篇人格深度长文，每篇 5000 字 + 名人案例 + 实战指南

### 关键数据指标(V1 上线后追踪)

| 指标 | 目标 | 备注 |
|---|---|---|
| 完测率 | ≥ 65% | 进入答题页到生成报告页的转化 |
| 分享率 | ≥ 25% | 生成报告到点击分享/截图的转化 |
| 公众号关注转化率 | ≥ 8% | 完测用户中关注公众号的比例 |
| 单次测评成本 | ≤ ¥0.5 | 含投流时的获客成本 |

## 七、商业化路径(已修正)

**V1.0 PRD 设计的路径(已废弃)**：
- ~~V2 高级报告付费 / V3 AI 分析 / V4 接入交易记录 / V5 社区~~ → 测评类产品不支撑这种深度商业化

**V1.1 修正后的路径**：

| 版本 | 核心动作 | 商业价值 |
|---|---|---|
| V1 | 免费测评 + 公众号引流 | 0 直接收入，目标是获客 |
| V1.5 | 互测功能、名人对照延伸 | 提升 K 值，扩大流量池 |
| V2 | 接入「量化新手村」的策略复盘内容、TradeCheck 工具 | 引流到主产品(明日当空) |
| V3 | 输出"你的人格 × 当前市场环境"的月度配置建议(不涉及个股) | 知识付费(¥99/年) |
| V4 | 私募候选人筛选工具(B 端，发给潜在 LP/合伙人) | 服务终极目标"开私募"的人脉沉淀 |

**核心原则**：TradeDNA 自己不背 KPI，它只为「量化新手村」IP 和私募路径输血。

## 八、合规红线

- ✅ 全程不涉及个股评价、买卖建议、目标价
- ✅ 不做业绩展示、不做收益承诺
- ✅ 推荐"交易体系"≠ 推荐"交易策略"，前者是方法论分类，后者才是合规敏感
- ✅ 名人对照仅作为人格映射，不暗示"跟随其策略即可盈利"
- ✅ 首页 + 报告页底部固定免责声明

- 纯前端实现，无后端依赖
- 答案 + 结果存 localStorage，不上传(降低合规复杂度)
- (V1.5 规划)接入埋点 SDK 追踪关键漏斗

## 十一、不做清单

> 防止精力分散，明确边界。

- ❌ 不做后端账户系统(用户标签靠公众号承接)
- ❌ 不做积分/任务/打卡机制(增加复杂度不增加价值)
- ❌ 不做社交关系链(微信生态本身已足够)
- ❌ 不做付费会员(参考 [project_strategy_package_sale])
- ❌ 不做"高级报告 ¥9.9"这类小额变现(损害专业感)
- ❌ 不做"AI 解读你的交易"承诺(技术不到位前别画饼)
