## 16 人格速查

### 理性稳态象限（RT**）—— 研究 + 稳态

#### RTCD · 量化工程师
- **画像**：把交易当工程系统，数据信徒，规则至上
- **适配**：量化交易、ETF 轮动、套利、趋势跟随
- **风险**：过度优化、错过情绪行情、参数微调陷阱
- **大师群**：西蒙斯 / 大卫·肖 / 邓晓峰

#### RTCF · 时间合伙人
- **画像**：深度研究+长期持有，"不动"是天赋而非无能
- **适配**：价值投资、行业投资、长期持有、红利策略
- **风险**：价值陷阱、爱上股票、长期掩盖该卖
- **大师群**：巴菲特 / 查理·芒格 / 段永平

#### RTGD · 数据狙击手
- **画像**：用纪律约束进攻，进攻+纪律罕见组合
- **适配**：量化主动权益、趋势跟随、事件驱动、多策略组合
- **风险**：连胜后失控（头号死神）、为了不闲着出手
- **大师群**：肯·格里芬 / 王琛 / 王亚伟 / 冯成毅

#### RTGF · 市场猎鹰
- **画像**：研究+热点共振，idea 永远比执行多
- **适配**：事件驱动、主题投资、行业轮动、成长股
- **风险**：什么都做、换股频繁吃复利、追二阶热点
- **大师群**：史蒂夫·科恩 / 邱国鹭 / 管大宇

### 理性敏感象限（RE**）—— 研究 + 情绪化

#### RECD · 觉醒计算师
- **画像**：所有反思圣经的作者，痛苦+反思=进步
- **适配**：趋势跟随（深度研究型）、价值投资、宏观对冲
- **风险**：失眠、反思变反刍、研究深度反成持仓死扛理由
- **大师群**：雷·达里奥（早期爆仓后涅槃）/ 青泽 / 霍华德·马克斯 / 谢治宇

#### RECF · 孤勇守仓人
- **画像**：公募一哥/女神的精神画像，研究勤奋+信念易动摇
- **适配**：行业深度投资、价值成长、主题投资、长期持有
- **风险**：信念突然崩塌、重仓单赛道、研究勤奋变焦虑性确认
- **大师群**：张坤 / 葛兰 / 刘格菘

#### REGD · 孤狼猎手
- **画像**：A 股最孤独的格，逆向重仓+严守纪律
- **适配**：价值投资（逆向方向）、深度成长股、宏观对冲（重仓型）
- **风险**：逆向变固执、单一持仓过重、孤独耗尽纪律电量
- **大师群**：德鲁肯米勒 / 冯柳 / 林园

#### REGF · 宏观狙击手
- **画像**：格局最大也最难落地，看时代+押产业+灵活进攻
- **适配**：宏观对冲、行业轮动、主题投资、成长股（产业大势）
- **风险**：风口幻觉、灵活=体系不稳、顶部下不来
- **大师群**：索罗斯 / 叶庆均（敦和）/ 梁瑞安（凯丰）/ 但斌 / 张磊

### 跟随稳态象限（XT**）—— 跟外部 + 稳态

#### XTCD · 复利守夜人
- **画像**：A 股最不性感的人格，"什么都不做"是稀缺天赋
- **适配**：宽基指数定投、红利策略、资产配置（股债平衡）
- **风险**：忍不住主动、选错指数、长期失效（日经/上证教训）
- **大师群**：约翰·博格 / 戴维·斯文森 / 沃尔特·施洛斯

#### XTCF · 灯塔守望者
- **画像**：跟随主流叙事+长期持有，"信念派"考验心力
- **适配**：主流价值投资、品牌投资、长期持有、ETF 主流宽基
- **风险**：主流叙事崩塌、对基本面敏感度低、伪共识接盘
- **大师群**：彼得·林奇 / 约翰·邓普顿 / 李录 / 赵丹阳

#### XTGD · 未来远征军
- **画像**：押注产业+押注时代，"押注未来"的浪漫与代价
- **适配**：成长股、科技股、创新产业、ETF 主题
- **风险**：押错赛道、估值钝感、单一产业重仓
- **大师群**：凯西·伍德 / 雷军 / 傅鹏博 / 朱少醒

#### XTGF · 风口骑士
- **画像**：期货圈最硬核，多年熬一波，一波吃十年
- **适配**：趋势跟随（CTA）、宏观对冲、商品期货
- **风险**：等不到那一波强行操作、杠杆放大、恋战不退
- **大师群**：吉姆·罗杰斯 / 林广茂（棉花哥）/ 葛卫东 / 傅海棠

### 跟随敏感象限（XE**）—— 跟外部 + 情绪化

#### XECD · 短线刺客
- **画像**：盘感+纪律双系统，A 股短线生存率最高的一格
- **适配**：超短交易（T+0）、龙虎榜跟单、打板（小仓位）
- **风险**：盘感失灵、交易成本吃复利、健康成本极高、年龄红利
- **大师群**：杰西·利弗莫尔（早期）/ 尼古拉斯·达瓦斯 / 孙哥（孙国栋）

#### XECF · 迷雾行者 ⚠️ 特殊格
- **画像**：A 股 70% 散户的真实画像，普通跟风散户原型
- **适配**：—— 任何特定体系都低匹配，因为你"没有体系"
- **风险**：慢性失血无声出局、"下次一定"循环、错觉式进化
- **大师群**：**无（留空）** —— 成功大师都从这格进化离开了
- **特殊处理**：结果页用警醒文案替代导师卡，引导用户选择进化方向

#### XEGD · 龙头战将
- **画像**：A 股最危险也最性感的格，跟情绪+龙头+重仓+纪律
- **适配**：龙头战法、打板、情绪周期、超短交易
- **风险**：连胜后失控、二线票诱惑、熊市无效、重仓单票
- **大师群**：徐翔（泽熙）/ 章建平 / 赵老哥 / 退学炒股 / 炒股养家 / 陈小群

#### XEGF · 焰火追猎者
- **画像**：16 格里爆发力最强也最危险，差一个 D 就是 XEGD
- **适配**：妖股交易（短期）；任何长线/系统化均不适配
- **风险**：纪律是事后回忆出来的、重仓无上限、上帝模式、报复性交易
- **大师群**：杰西·利弗莫尔（晚期）/ 张文军 / 宁波桑田路·陆大宝 / 欢乐海岸 / 佛山无影脚

## 群体分布预估（A 股散户）

| 象限 | 占比预估 | 主要人格 |
|---|---|---|
| RT** 理性稳态 | 8-12% | RTCF（基民转化）、RTCD（程序员转化）|
| RE** 理性敏感 | 10-15% | RECD、RECF（公募基民） |
| XT** 跟随稳态 | 12-18% | XTCD（指数定投派）、XTCF（白马党） |
| **XE** 跟随敏感** | **60-70%** | **XECF（迷雾行者，最大单格）** |

XECF 单格占比预估 **30-40%** —— 这是产品定位的核心数据基础。

## 命名规范（V3 决议）

- 不用"神/宗师/战神"等明显高帽词
- 保留有画面感、有故事性的中性/史诗感词汇
- 保留功能描述性词汇（工程师、合伙人、猎手、狙击手）
- 让用户**愿意发朋友圈说"我是 XX"**——既不夸大也不丢脸

### V2→V3 命名变更记录

| DNA | V2 提案 | V3 终稿 | 调整理由 |
|---|---|---|---|
| RTCD | 量化宗师 | 量化工程师 | "宗师"是高帽 |
| RTCF | 时间布道者 | 时间合伙人 | "布道"宗教感太强 |
| REGD | 孤狼战神 | 孤狼猎手 | "战神"是高帽 |
| REGF | 宏观指挥官 | 宏观狙击手 | 用户明确反馈"狙击手"优于"指挥官" |
| XEGD | 龙头战神 | 龙头战将 | "战神"是高帽 |
