# 公众号图文发布版 · 排版指南

> 这个目录的 16 个 md = 原始 md + 统一文末 CTA,可直接送进 markdown→公众号的排版工具
> 16 篇全部上线后,所有 DNA 码关键词回复都能绑定对应图文

---

## 一、推荐工具:墨滴 mdnice

**地址**:https://mdnice.com

**为什么用它**:
- 免费,无登录,无水印
- 专门为公众号设计,有多套现成主题(简约 / 商务 / 文艺)
- 一键复制到公众号编辑器,样式完整保留
- 支持代码块、引用、列表、加粗、emoji 等所有 markdown 语法

**替代品**(任选其一):
- 秀米 https://xiumi.us (老牌排版,有图文模板,可商用)
- 迹忆客 https://www.jiyik.com/md (同类)

## 二、操作流程(每篇 5-10 分钟)

### 1. 在 mdnice 排版

1. 打开 https://mdnice.com
2. 把发布版 md 文件的内容**全部粘贴到左侧编辑区**
3. 上方菜单「主题」→ 选一个适合金融/商务的主题(推荐:**「兰青」「锦书」「兰陵笑笑生」**)
4. 上方菜单「字体」→ 选「**思源黑体**」或「**默认**」(避免中英文混排不齐)
5. 上方菜单「字号」→ **15px**(公众号默认)
6. 点右上角「**复制**」按钮 → 提示「已复制到剪贴板」

### 2. 粘贴到公众号

1. 公众号后台 → **内容管理** → **图文消息** → **新建图文**
2. 在编辑区里 `Ctrl + V` 粘贴(样式会保留)
3. 上方填写:
   - **标题**:取 md 第一行 `# ...` 的内容(去掉 `#`),如:
     - XEGD → "你是 A 股最危险也最性感的人格——能活下来的不到 5%"
   - **作者**:`交易人格实验室`
   - **摘要**:取 md 第二行的人格定义,如 "RTCD · 量化工程师 · 数据信徒派"
   - **封面**:用 `brand/logo_avatar_512.png` 或者各人格的对照大师头像
   - **原创**:✅ 勾选(申请原创保护)

### 3. 保存到素材库(不要群发)

- 点「**保存**」(右上角)→ 进入素材库
- ⚠️ **不要**点「群发」——订阅号 1 天只能群发 1 次,这样会用掉今天的群发额度
- 16 篇逐一处理,每天慢慢做也行,不影响

### 4. 绑定到关键词回复

1. 公众号后台 → **自动回复** → **关键词回复**
2. 添加规则(或编辑已有规则):
   - 规则名:`DNA-XEGD`
   - 关键词:`XEGD`,**全匹配**
   - 回复类型:选「**图文消息**」(不要选「文字」)
   - 从素材库选刚才保存的那篇
   - 保存
3. 重复 16 次,完成所有 DNA 码
