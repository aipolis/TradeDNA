# 公众号「交易人格实验室」· 自动回复配置清单

> 共 18 条:16 个 DNA 码 + 2 个特殊关键词(`全集` / `测评`)
> 配置一次约 5 分钟,以后改文案直接编辑回复内容即可

---

## 一、配置步骤

1. 登录 [mp.weixin.qq.com](https://mp.weixin.qq.com)
2. 左侧菜单:**自动回复** → **关键词回复**
3. 点「**添加规则**」
4. 规则名随意填(如 `DNA-RTCD`)
5. 关键词:粘贴下方关键词(如 `RTCD`),匹配方式选「**全匹配**」
6. 回复内容:粘贴下方对应文案
7. 保存
8. 重复 18 次,完成所有规则

## 六、文案设计原则(以后改文案参考)

- **首行**:DNA 码 + 人格名(身份认同)
- **核心标签**:tag(品牌词)
- **你是谁**:1 句话画像(诊断感)
- **优势 / 陷阱 / 大师 / 适合体系**:结构化信息,便于扫读
- **结尾**:固定 2 行 CTA(全集 PDF + 小程序入口)

合规底线:**不出现具体个股代码、不暗示按某策略可获益、不承诺收益**。
