# TradeDNA 抖音脚本框架 V3

> 2026-06-24 V3 重写。基于四维改造（信息 R/X × 情绪 T/E × 风险 C/G × 执行 D/F）+ 多导师人格库。
> V1 框架已废弃（旧维度+旧人格已失效）。
> 配套 16 篇深度解析（公众号引流回填）见 `人格详解_*.md`。

---

## 内容设计原则（V3 修订）

- **禁言后第二轮投放策略**：前 2 周仍避免明显诱导，公众号引流后置到主页/评论区，不直接放二维码
- **引流话术统一**：
  - "主页有个测试，5 分钟找到你的交易人格"
  - "评论区告诉我你的 DNA 码，我给你回对应深度报告"
- **每条 30-60 秒**，前 3 秒钩子决定生死
- **人设**：不是导师，是"陪你看清自己的散户同行"
- **V3 流量抓手**：争议性配比——同一格塞海外传奇+A 股本土+游资天团形成讨论
- **合规底线**：不评个股、不承诺收益、不诱加微信、不暗示跟单

## 运营观察期备忘（变现方向探测）

> V3 阶段策略不变：**先做流量，变现边走边看**。
> 跑流量期间留意核心信号：**用户测完之后，主动问什么**。

| 用户反馈类型 | 暗示的变现方向 | 备注 |
|---|---|---|
| 大量问"我是 XXXX，该用什么工具/策略" | 工具变现（明日当空 / TradeCheck）| 用户已认可"人格×工具"逻辑 |
| 大量问"能不能加你/聊聊我的交易" | 陪伴社群（知识星球）| 合规边界，严禁个股建议 |
| 多是"哈哈我是 XXXX"娱乐反馈 | 流量广告（粉丝 5k+ 接软广）| 用户消费内容而非求解 |
| 高净值/机构口吻私信 | B 端（PRD V4）| 罕见但有标记价值 |

记录方式：每周末复盘评论区，追加 `YYYY-WW: A=X% / B=X% / C=X% / 备注`，3 个月看趋势决定路径。
